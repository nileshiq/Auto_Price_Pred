# 🚗 Automobile Dataset Analysis

## 📌 Overview

This project is based on the Automobile Dataset, which contains detailed information about different car models, their specifications, and pricing.

The dataset is widely used for data analysis and machine learning tasks. The main goal is to understand how various features of a car influence its price.

---

## 📊 Dataset Description

The dataset includes the following features:

- Company / Brand (Audi, BMW, etc.)
- Fuel Type (Gas, Diesel)
- Aspiration (Standard, Turbo)
- Number of Doors
- Body Style (Sedan, Hatchback, Convertible, etc.)
- Drive Wheels (FWD, RWD, AWD)
- Engine Type
- Number of Cylinders
- Engine Size
- Fuel System
- Horsepower
- Peak RPM
- City Mileage
- Highway Mileage
- Price (Target Variable)
